Универсальный провайдер abilities для WordPress MCP Adapter. Мгновенно открывает управление контентом и WooCommerce для AI-агентов на любом WordPress-сайте.

== Description ==

**MCP Abilities Provider** — плагин-мост между вашим WordPress сайтом и AI-агентами (Claude, ChatGPT, Cursor, VS Code и др.) через [Model Context Protocol](https://modelcontextprotocol.io/).

Плагин решает главную проблему: после установки MCP Adapter abilities не видны AI-клиентам, потому что ни одна из них не помечена как `mcp.public`. Этот плагин:

1. **Открывает core abilities** (get-site-info, get-user-info, get-environment-info) для MCP
2. **Регистрирует 40+ abilities** для полноценного управления сайтом и магазином через AI

= Доступные abilities =

**Посты:**
* Получить список постов с фильтрацией
* Получить пост по ID (с полным контентом)
* Создать пост
* Обновить пост
* Удалить пост

**Страницы:**
* Получить список страниц
* Получить страницу по ID
* Создать страницу
* Обновить страницу
* Удалить страницу

**Медиа:**
* Получить список медиафайлов
* Обновить метаданные медиафайла

**Таксономии:**
* Получить категории / создать категорию
* Получить метки / создать метку

**Пользователи:**
* Получить список пользователей

**Комментарии:**
* Получить комментарии
* Модерировать комментарий

**Настройки:**
* Получить настройки сайта
* Обновить настройки

**Плагины:**
* Получить список установленных плагинов

**Меню:**
* Получить навигационные меню с элементами

**WooCommerce — Товары (если WooCommerce активен):**
* Получить список товаров с фильтрацией по категории, типу, цене, наличию, SKU
* Получить товар по ID (вариации, атрибуты, галерея, рейтинг)
* Создать товар (простой, вариативный, групповой, внешний)
* Обновить товар
* Удалить товар

**WooCommerce — Категории и метки товаров:**
* Получить категории товаров (дерево с изображениями)
* Создать категорию товаров
* Получить метки товаров
* Создать метку товаров

**WooCommerce — Заказы:**
* Получить список заказов с фильтрацией по статусу, дате, клиенту
* Получить заказ по ID (товары, адреса, оплата, заметки)
* Обновить статус заказа с заметкой

**WooCommerce — Купоны:**
* Получить список купонов
* Создать купон (процент, фиксированная скидка, бесплатная доставка, лимиты)

= Требования =

* WordPress 6.9+ (с Abilities API в ядре)
* [MCP Adapter](https://github.com/WordPress/mcp-adapter) v0.4.0+
* PHP 8.1+
* WooCommerce 8.0+ (опционально, для WooCommerce abilities)

= Безопасность =

* Все abilities защищены через `permission_callback` с проверкой прав (`current_user_can`)
* Входные данные валидируются через `sanitize_text_field()`, `absint()`, `wp_kses_post()`
* Деструктивные операции (удаление) требуют повышенных прав
* WooCommerce abilities используют нативные WC API (WC_Product, WC_Order, WC_Coupon)

== Installation ==

1. Убедитесь, что установлен WordPress 6.9+ и плагин MCP Adapter
2. Скачайте и распакуйте `mcp-abilities-provider` в `wp-content/plugins/`
3. Активируйте плагин через «Плагины» в админке
4. Abilities автоматически станут доступны для MCP-клиентов
5. WooCommerce abilities появятся автоматически, если WooCommerce активен
